# You Deserve

一个写给双非鼠鼠的八股学习站。

如果你也经历过这些时刻：

- 打开招聘软件，看到一堆 “985/211 优先”
- 简历投出去，已读不回像空气
- 学八股学到怀疑人生，越学越觉得自己什么都不会
- 刷题刷到凌晨，第二天又被一句 “项目太普通了” 干沉默

那这个小站，大概率就是写给你的。

## 鼠鼠宣言

我是一个双非鼠鼠。

没有很闪的学历背景，没有天降的内推礼包，没有开局自带大厂光环。  
我只是一个会焦虑、会怀疑自己、会在深夜默默补八股、补项目、补基础的普通人。

但普通，不代表要认输。

这个项目的出发点很简单：

**如果现实一开始没有给我们太多筹码，那我们就自己一张一张攒。**

八股不是终点，但它是门票。  
项目不是神话，但它是证据。  
表达不是表演，但它决定你会不会被看见。

这个站想做的事，就是把那些零散、难记、容易越学越乱的知识点，整理成一套更清晰、更能复习、更适合双非同学上岸节奏的内容。

不是为了把人训练成背书机器。  
而是希望你在面试里，至少能更稳一点、更清楚一点、更像一个真的做过思考的人。

## 这个项目有什么

目前它主要提供：

- 面向后端/Java 方向的八股题整理
- 每题更完整的 `详细讲解`
- 每题一份可视化图解
- 更适合复习的知识结构和页面展示

这次已经完成了一轮比较重的内容增强：

- 共 `400` 道题
- 每题都补了 `详细讲解`
- 每题都补了图解数据
- 全量详解都做过长度校验

也就是说，这不是一个“先摆着，后面再补”的半成品坑位，而是一套已经能直接打开学、打开背、打开复盘的内容站。

## 为什么叫 You Deserve

因为很多双非同学，太容易先被环境否定。

会先默认自己不够好。  
会先觉得别人的起点更高。  
会先把“拿到机会”当成一种运气，而不是一种自己也配拥有的结果。

但我想把这个名字放在这里：

**You Deserve.**

你值得一个更像样的机会。  
你值得被认真看见。  
你值得一次靠自己一点点打上去的上岸。

不是因为谁安慰你。  
而是因为你真的已经在很努力地往前走了。

## 适合谁

这个项目尤其适合：

- 双非本科、普通背景、想冲后端/Java 的同学
- 基础零散，学了很多但说不出来的人
- 想系统复习面试题，而不是到处抄碎片答案的人
- 想把“背八股”变成“理解 + 表达 + 复盘”的人

如果你已经很强，它也许只是一个普通资料站。  
但如果你还在一点点往上爬，我希望它能成为那个“至少别让我再乱学了”的落脚点。

## 本地启动

```bash
npm install
npm run dev
```

默认是一个 Next.js 项目。

## 项目结构

```text
content/questions/           题目内容
content/visuals/             图解数据
scripts/                     批处理和内容增强脚本
src/app/                     Next.js 页面
src/components/              页面组件
```

## 一点真心话

这个项目不是在说“双非也能逆袭”这种轻飘飘的大词。

现实不会因为你努力就自动温柔。  
面试官不会因为你辛苦就直接给你 offer。  
很多门槛也不会突然消失。

但即便这样，还是可以继续往前拱。

一点点补基础。  
一点点练表达。  
一点点把项目讲明白。  
一点点让自己从“完全没底”变成“至少这题我能说清楚”。  

双非鼠鼠不是传奇主角。  
但鼠鼠也能一步一步，把自己的路刨出来。

如果你现在正卡在某个阶段，先别急着判自己死刑。  
先多学一道题，多补一个项目细节，多过一次面试复盘。

很多上岸，不是突然开窍。  
只是有人咬着牙，多坚持了一段。

愿这个仓库，能陪你把那一段走完。
