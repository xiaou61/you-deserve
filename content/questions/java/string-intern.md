## 一句话结论

`intern()` 会返回字符串常量池中对应字符串的引用，如果常量池中没有，则尝试把当前字符串加入常量池。

## 通俗解释

字符串常量池像公共字典。相同的词没必要每个人都复印一份，大家指向字典里的同一个词条即可。

## 面试回答

Java 为了复用字符串，会维护字符串常量池。字面量字符串通常会进入常量池。

调用 `intern()` 时：

- 如果池中已有相同内容的字符串，返回池中引用。
- 如果没有，则把字符串放入池中或记录引用，再返回池中引用。

它可以减少重复字符串内存占用，但不应该在业务里盲目调用，因为常量池本身也占内存，滥用可能带来性能和内存问题。

- 不要把 intern 当成默认优化手段。
