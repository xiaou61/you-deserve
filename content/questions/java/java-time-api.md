## 一句话结论

Date 把时间点、格式化和时区语义混在一起，java.time 则拆成 LocalDate、LocalDateTime、Instant、ZonedDateTime 等明确类型。

## 通俗解释

旧 Date 像一个什么都装的箱子，java.time 像分类收纳盒。

## 面试回答

可以从这几层回答：

- LocalDate 表示日期，LocalTime 表示时间，Instant 表示时间戳。
- 对象不可变，天然线程安全。
- 时区使用 ZoneId 和 ZonedDateTime 显式表达。
- 格式化用 DateTimeFormatter，比 SimpleDateFormat 更安全。
